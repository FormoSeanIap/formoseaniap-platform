如同之前所提及的，這部作品神奇的地方，在於其主題的模糊性。

故事發生在魔王已經被勇者打倒了的世界裡，在這個和平已然到來的世界裡，明確的目標並不存在，角色也不真的擁有某個偉大的動機，僅僅是日復一日地在日常生活的推進中回憶起各種過去的瑣事。

我一直在想，這樣類型的作品憑什麼能夠在當代社會掀起這麼大的熱潮呢？這又與我們過去熟悉的，擁有明確目標和角色成長的作品有什麼不同呢？

## 過去的作品

在過去典型的王道作品裡面，通常會伴隨著主角確立一個明確目標，透過各種事件獲得成長，最終達成目標的設計。

典型的作品當屬《火影忍者》(-ナルト- / NARUTO)、《海賊王》(ワンピース / ONE PIECE)、《死神》(BLEACH)、《獵人》(ハンター×ハンター / HUNTER×HUNTER)等等，而直到更晚近的作品中，仍不乏這樣子的設計，比如《鬼滅之刃》(鬼滅の刃 / Demon Slayer)、《咒術迴戰》(呪術廻戦 / Jujutsu Kaisen)等等。

這類劇情設計的好處在於方向明確，角色成長的軌跡清晰，觀眾也能夠在過程中感受到成就感和滿足感。縱使如《死亡筆記本》(デスノート / DEATH NOTE)或《烙印勇士》(ベルセルク / Berserk)這類非屬善惡分明的作品，也同樣有著明確的目標和角色成長的設計。

不過也隨著同類作品的推陳出新，開始有更多作品嘗試打破這樣子的框架。

比如《進擊的巨人》(進撃の巨人 / Attack on Titan)帶出了更多的「並非努力就必然成功」的現實感，而《鏈鋸人》(チェンソーマン / Chainsaw Man)則更有強烈的「目標本身也許可以很隨便」的感覺，此外大量出現的各種異世界轉生類型作品，則有許多主角一開始就獲得最強能力而無須成長的設定。

## 反映社會的作品

我認為，能夠在當代流行起來的作品，通常也都反映了當代社會的某些特質或需求。

我們通常都會期待在我們的努力之下，最終可以達成我們一開始所設立的目標，因此王道作品歷久彌新。

但也許在社會還處在有成長空間的階段時，努力即有所成的故事即使在現實中也時有所聞，因此這樣子的敘事也同時可以滿足觀眾的期待。

\
而一直到某個時間點，社會成長逐漸飽和而趨緩的當下，努力不一定可以看到明確成果的現實感開始普遍了起來，那些宣揚努力就可以獲得明確成果的作品開始變得似乎有些刺耳，如同長輩說教感一樣。

因此，沒有好的結果也沒有關係，或我們在現實中有強烈的無力感，但看到在異世界中直接獲得最強能力的主角，或是看到主角在努力之後也不見得能夠達成目標的作品，反而就變得有一種莫名的療癒感了吧。

不過，隨著時代的演進，我們不只不再汲汲營營追求明確的目標和進步，我們還更進一步地轉頭去注意我們那看似毫不起眼的人生，在仔細品味日常生活的過程中，獲得療癒自己的能力。

這是我認為《葬送的芙莉蓮》這樣子的作品能夠在當代社會掀起熱潮的原因。

## 瑣碎的日常

這部作品從一開始，就不斷強調各種日常生活上的瑣事。

比如說，作品幾乎從來沒有講述過勇者打倒魔王的細節，宛如這樣的事蹟還遠遠比不上他們在旅途中的無聊瑣事一樣。

\
如同在一級魔法使測驗篇章[1]中，北部魔法隊隊長威亞貝爾(ヴィアベル / Wirbel)提到對於勇者欣梅爾的印象。
那些攻破複雜迷宮，擊敗巨龍，或是與強大魔族對決的偉大傳說，小時候的他確實對此深受吸引。

但也如同他所述，偉大的傳說與他們的日常其實沒有什麼關係，而所有人光是守護自身的日常生活就已然竭盡全力。

因此，真正為村民所記得的事實，都是一些雞毛蒜皮的瑣事，但神奇的是，這些瑣事卻是如此珍貴，而那些村民反而是在分享這些微不足道的小事時，才打從心裡覺得開心。

## 無趣的人生

也許確實是如此吧。

隨著社會發展逐漸走到某種飽和的階段，搭著社會發展的列車，再憑藉自身努力而獲得成功的故事，也逐漸變得不切實際了起來

同樣的努力於當代已然無法創造如同過去數十年一樣明確的成果，所謂成功人生的故事也不再具有那麼高的說服力，人生目標的轉向似乎也理所當然。

\
我們開始承認，自己的人生就是不上不下，半推半就。沒有什麼偉大的成就，但也就是活得下去而已。

但就算是微不足道的人生，就算是無聊的日常瑣事，也可以是有意義的，也仍然可以是值得我們去珍惜的，甚至是可以讓我們感到開心的。

從社會發展的脈絡來看，這似乎是合情合理的。我們也許仍有目標，但我們也可以享受過程；而若能享受過程，結果是如此也許也並非如此重要。

不過，能夠體悟這樣的人生觀，並實踐在自己的生活中，充分意識到用心感受每天的日常生活的意義，仍然是一件不簡單的事情呢。

\
也許我們都需要那個可以變出花園的魔法。

## 重新認識一個人

我認為，威亞貝爾提到對於欣梅爾的印象的這段話，對於我來說也是非常有啟發性的。

我們通常是怎麼認識一個人的呢？

我想，最一開始都是透過一個人身上的標籤來認識他的吧，比如說他的職業、他的成就、他的外貌等等。

\
欣梅爾最大的標籤可能就是「打倒魔王的勇者」，也許還附加有各種不同的傳說與故事。

但我想，在社會變動快速的當代，也許我們比以前更缺乏真正認識一個人的機會。

對於村民來說，比起一個「打倒魔王的勇者」，他們真正認識的，其實是一個會幫他們修理房子的普通人。

但也許這才是真正的認識一個人的方式吧，透過日常生活的相處，透過那些看似微不足道的瑣事，去認識一個真正的他，而非只是他身上的標籤了。

\
確實，標籤可能是認識一個人的開始，但如果我們只停留在標籤的表面，那麼我們就永遠無法真正認識一個人了。

我也希望別人不是透過標籤認識我，而是認識一個真正的我，而非只是某些成果，更重要的是我如何成為這樣的人的過程。

我想，這是在當代社會尤為重要的事情了吧。
